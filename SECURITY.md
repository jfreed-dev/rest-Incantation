# Security Policy

## Supported Versions

Security fixes are provided for the latest code on the `main` branch.

| Version | Supported |
| ------- | --------- |
| main    | Yes       |
| others  | No        |

## Reporting a Vulnerability

Please report vulnerabilities via GitHub Security Advisories if available.
If you cannot use advisories, open a GitHub issue with minimal details and
request a private channel for follow-up.
